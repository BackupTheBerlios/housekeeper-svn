How to start Housekeeper:

Windows:

  1. Use the Windows Explorer to navigate to the directory where you have installed Housekeeper.
  2. Double-click on "housekeeper.jar" to start the application. If this doesn't work, double-click on "housekeeper.bat"

Linux/Mac OS X/Unix

  1. Using you favourite file manager, navigate to the directory where you have installed Housekeeper.
  2. Double-click on "housekeeper.jar" to start the application. If this doesn't work, double-click on "housekeeper.sh".
